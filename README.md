# Get Any Scent Shopify Theme

This is a Shopify theme rebuilt from the provided theme export and redesigned using the visual direction from `masnoori9090/getanyscent`.

## What changed

- Replaced the default homepage with custom editorial sections inspired by the React reference.
- Added a shared luxury design system with cream, forest, and gold styling.
- Upgraded header, product cards, footer, and product page presentation through a dedicated stylesheet.
- Kept the Dawn-compatible theme structure so the folder can still be uploaded as a Shopify theme ZIP.

## Upload to Shopify

1. Zip the contents of this folder.
2. In Shopify Admin, go to `Online Store > Themes`.
3. Click `Add theme > Upload zip file`.
4. Upload the generated ZIP.

## GitHub

`gh` is not installed in this environment, so the remote GitHub repository was not created automatically.
The project is prepared as a standalone local git repository and can be pushed with:

```bash
git remote add origin <your-new-github-repo-url>
git push -u origin main
```
